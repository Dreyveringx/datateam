Demo de Proyecto con Spring Boot
Este proyecto es un demo de una aplicación web donde comenzamos con una estructura básica en HTML, pero nos centraremos principalmente en el desarrollo del backend utilizando Spring Boot. En esta primera parte, configuraremos la conexión a la base de datos y crearemos un sistema de login básico.

Tecnologías Utilizadas
Frontend: HTML básico, css basico
Backend: Spring Boot
Base de Datos: PostgreSQL
Framework de Backend: Spring Security 
Estructura del Proyecto
Frontend:
Un archivo básico index.html con estilo css basico, para visualizar la interfaz.
Backend:
Spring Boot para crear la API REST.
Conexión a la base de datos con JPA y Hibernate.
Implementación de un sistema de login básico con Spring Security.
